# Global-Store-Sales-Analysis-Excel-PowerBI
![Excel](https://img.shields.io/badge/Tool-Excel-green)
![Power BI](https://img.shields.io/badge/Tool-Power%20BI-yellow)
![Project Type](https://img.shields.io/badge/Type-Business%20Intelligence-blue)
This project presents an end-to-end sales performance analysis of a global retail store dataset using Microsoft Excel for data preparation and Power BI for interactive dashboard development.
The objective was to simulate a real-world business intelligence workflow — from raw data cleaning to executive-level dashboard reporting.
Project Workflow

1️. Data Cleaning & Preparation (Excel)
 Removed duplicates
 Handled missing values
 Standardized date formats
 Created calculated columns (Profit %, Shipping Days)
 Structured dataset for BI modeling
 Data Modeling & Visualization (Power BI)

2. Built relationships
  Created DAX measures
  Designed interactive dashboard
  Applied conditional formatting
  Created business KPI cards

3.Tools & Technologies Used
 Microsoft Excel (Data Cleaning & Preprocessing)
 Power BI (Dashboard & DAX)
 Data Modeling
 Business Intelligence Concepts
 KPI Analysis

4.Dashboard Structure
  Page 1 — Executive Overview
  Total Sales
  Total Profit
  Profit Margin %
  Revenue Trend
  Sales & Profit by Region
  Segment Distribution
  
  Page 2 — Product Intelligence
   Category vs Sub-Category Analysis
   Top 10 Products
   Loss-Making Products
   Discount vs Profit Impact
   
   Page 3 — Operations & Logistics 
    Shipping Mode Performance
    Average Shipping Days
    Order Priority Analysis
    Shipping Cost Impact
   
  5. Key Business Insights
    High discount rates significantly impact profitability
    A small number of products generate majority of revenue
    Some regions show high sales but lower margins
    Shipping mode influences delivery performance

  6. Business Value
    This dashboard enables management to Optimize discount strategy
    Identify loss-making products
    Improve regional sales strategy
    Enhance operational efficiency
